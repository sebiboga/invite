INVITAȚIE

Dragă Diana,

peviitor.ro este un proiect open source — codul sursă este public, oricine poate contribui, iar fiecare loc de muncă adus în platformă face parte dintr-un efort colectiv. Construim gratuit cea mai completă bază de date cu oportunități de carieră din România, iar asta nu ar fi posibil fără oameni care se implică voluntar, fără remunerație, din pură pasiune pentru misiunea noastră.

Ecosistemul peviitor.ro este format din mai multe componente — scrapere, API, interfață de căutare, validare, testare — fiecare dezvoltată de voluntari dedicați. Avem nevoie de cineva care să vadă imaginea de ansamblu și să se asigur că totul se leagă.

Rolul pe care ți-l propunem este acela de Project Manager pentru întreaga echipă de voluntari. Vei avea în responsabilitate toate echipele și toate taskurile lor — nu doar scraping-ul, ci tot ce înseamnă peviitor.ro. Tu ești cea care știe cine la ce lucrează, cât timp mai este necesar până la finalizare și când vor fi livrate componentele care compun ecosistemul.

Vei urmări activitatea voluntarilor în GitHub Projects, pe un dashboard de tip Big Picture — un roadmap care îți dă vizibilitate completă asupra întregului proiect. De acolo vei ști exact stadiul fiecărei componente, cine are blocked-uri și ce trebuie deblocat pentru ca livrările să meargă înainte.

Comunicarea se face pe Discord, iar organizarea întregului efort — pe GitHub: issues, pull requests, projects, roadmap, totul este acolo, transparent și urmăribil.

Acest proiect trăiește datorită oamenilor care cred că tehnologia poate simplifica modul în care românii își găsesc un loc de muncă. Prin implicarea ta, contribuția devine mai mult decât voluntariat — devine un impact real, vizibil, care rămâne.

Cu încredere și recunoștință,

Sebastian Boga
